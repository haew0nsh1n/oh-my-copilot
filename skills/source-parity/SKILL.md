# Source Parity

Compare the OMC `src/` runtime families against the OMP Python, CLI, Markdown, bridge, and hook surfaces.

OMP CLI: `omp source-parity [path/to/oh-my-claudecode/src]`.

Use this skill to distinguish implemented, adapted, partial, and missing source-level parity slices before choosing the next implementation target.
